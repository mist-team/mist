template<typename M, typename T>
class tagged_section< M, T >

Interval-treeのためのタグ情報付き区間を表すクラス

区間の両端を表す2つの値（最大値，最小値）と区間を識別するためのタグ情報を持つ．

注意
[ min, max )で表される区間を対象とする．
引数
M… 区間の最大値，最小値の型
T… 区間に付けられるタグ情報のデータ型
