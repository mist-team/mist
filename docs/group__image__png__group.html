説明

PNGファイルを読み込んで，MISTコンテナへ格納する． また，MISTコンテナからPNGファイルへの書き出しをサポート． 別途 http://libpng.org が提供するフリーのPNGライブラリが必要です．

次のヘッダをインクルードする
#include <mist/io/png.h>
