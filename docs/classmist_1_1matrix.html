説明

template<class T, class Allocator = ::std::allocator< T >>
class mist::matrix< T, Allocator >

行と列の要素数が可変の行列

  • CHECK_MATRIX_OPERATION マクロが1の場合は，行列演算の左辺と右辺で行列のサイズが適切であるかをチェックする
  • CHECK_ACCESS_VIOLATION マクロが1の場合は，行列の要素へアクセスする際に，範囲外かどうかを判定する
